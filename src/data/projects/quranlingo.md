#### Description
This is an application to improve understanding of the verses of the Quran using the Quiz method like the Duolingo application.

#### Role
UI/UX Designer

#### Technology & Frameworks
Figma and Canva.

#### Features:
- Quiz feature to test understanding of Quran verses.
- Multiple Choice Quiz Text
- Multiple Choice Quiz Audio
- Word Matching Text
- Word Matching Audio
