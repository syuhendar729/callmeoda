#### Description
This is a periodic table website application with an interactive display to find out the details of its cells and classification by group. I developed the front-end of this application with Bootstrap.

#### Role
Frontend Developer

#### Technology & Frameworks
JavaScript and Bootstrap.

#### Features:
- Display detailed information from a cell with a popup modal.
- Classification of elements based on groups.
