#### Invisible Maze Game

![Alt text](/projects/invisible-maze.jpeg "Optional title")


#### Description
This is an invisible maze game where players are required to memorize the walls of the maze in a short time, then the walls will disappear and the player must run the object to reach the finish.

#### Role
Frontend Developer

#### Technology & Frameworks
JavaScript, CSS, and ReactJS

#### Features:
- Makes the wall disappear within a specified time.
- Makes the player able to move with the arrow keys.
